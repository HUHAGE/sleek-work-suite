# 菜单管理功能

## 功能说明

在设置页面新增了"菜单管理"功能，允许用户自定义侧边栏菜单的显示和顺序。

## 主要特性

### 1. 菜单开关
- 每个菜单项后面都有一个iOS风格的开关按钮
- 开启：菜单在侧边栏中显示
- 关闭：菜单在侧边栏中隐藏

### 2. 菜单排序
- 每个菜单项右侧有上下箭头按钮
- 点击向上箭头：将菜单项向上移动一位
- 点击向下箭头：将菜单项向下移动一位
- 侧边栏菜单会按照设置的顺序显示

### 3. 持久化存储
- 所有菜单配置会自动保存到本地
- 重启应用后配置会自动恢复

## 技术实现

### 数据结构
```typescript
interface MenuConfig {
  id: string      // 菜单ID
  enabled: boolean // 是否启用
  order: number   // 排序顺序
}
```

### 核心组件
1. `MenuManagement.tsx` - 菜单管理组件
2. `settings.ts` - 设置状态管理（Zustand）
3. `Index.tsx` - 主页面，根据配置动态渲染菜单

### 工作流程
1. 用户在设置页面调整菜单配置
2. 配置通过Zustand store更新
3. store自动调用electron IPC保存到本地文件
4. 主页面监听配置变化，动态更新侧边栏显示

## 使用方法

1. 点击侧边栏底部的"设置"按钮
2. 在设置页面找到"菜单管理"部分
3. 使用开关控制菜单显示/隐藏
4. 使用上下箭头调整菜单顺序
5. 配置会自动保存，无需手动操作
